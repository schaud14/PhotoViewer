# PhotoViewer

A Picasa-like desktop photo viewer and organizer built with **Electron + React + TypeScript + MUI**.

## Features

- **Photo Library**: Scan folders, index images (JPEG, PNG, HEIC, RAW — DNG, NEF, ARW)
- **Virtual Albums**: Tag-based grouping, photos stay in original locations
- **Physical Albums**: Folder-backed albums that move files on disk
- **Tagging & Search**: Add tags, autocomplete, debounced search by tag/filename
- **Views**: Responsive grid (small/medium/large thumbnails), sortable list view
- **Photo Viewer**: Full-screen viewer with keyboard navigation, zoom, slideshow
- **Metadata Panel**: EXIF data, file info, editable tags
- **Trash**: Move to trash, restore to original location, permanent delete

## Getting Started

```bash
# Install dependencies
npm install

# Run in development mode
npm run dev

# Build for production
npm run build
```

## Development Notes

- **Database**: SQLite (via `better-sqlite3`), stored at `~/Library/Application Support/photo-viewer/data/photoviewer.db`
- **Thumbnails**: Generated by `sharp`, cached at `~/Library/Application Support/photo-viewer/.cache/thumbnails/`
- **Trash**: App-managed trash folder at `~/Library/Application Support/photo-viewer/trash/`
- **macOS Permissions**: The app may need Full Disk Access for scanning external drives
